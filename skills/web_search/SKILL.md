# Web Search

## Usage
When the user asks to:
- Search for current information
- Find recent news or events
- Look up specific topics or facts
- Research a question that requires up-to-date information

## Examples
- User: "What's the latest news about AI?"
- User: "Search for Python tutorial best practices"
- User: "Find information about climate change 2024"

## Constraints
- Use search engines responsibly
- Respect rate limits
- Only search for legitimate information requests
- Do not search for harmful or illegal content
